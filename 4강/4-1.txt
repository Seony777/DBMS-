CRUD: Create(생성), Read(조회), Update(갱신), Delete(삭제)
데이터베이스에서 자주 사용하는 데이터 조작(Data Manipuation)의 핵심

1. 데이터 삽입(Create)
INSERT INTO <테이블명> (컬럼1, 컬럼2, ....)
VALUES
(값1, 값2, ...)
(값1, 값2, ...)

2. 데이터 조회(Read)
SELECT 컬럼1, 컬럼2, ...
FROM <테이블명>
WHERE 조건;
- > WHERE로 특정 조건을 만족하는 행만 필터링 가능

3. 데이터 갱신(Update)
UPDATE <테이블명>
SET 컬럼1 = 새로운값1, 컬럼2 = 새로운값2, ...
WHERE 조건
- > UPDATE 뒤에 수정할 테이블명
- > SET 키워드 뒤에 변경할 컬럼과 새 값을 할당
- > WHERE 조건을 생략하면 테이블 전체 행이 수정되므로 주의!!

4. 데이터 삭제(Delete)
DELETE FROM <테이블명>
WHERE 조건;
-> DELETE FROM 뒤에 대상 테이블명
-> WHERE로 특정 행만 삭제 (조건을 생략하면 모든 행이 삭제되므로 주의!!)
